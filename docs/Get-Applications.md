# Get-Applications

## SYNOPSIS
List locally installed applications

## SYNTAX

```
Get-Applications [<CommonParameters>]
```

## DESCRIPTION
Query local registry for installed applications.

## EXAMPLES

### EXAMPLE 1
```
Get-Applications
```

Description
-----------
This will generate all installed applications on the local system.
