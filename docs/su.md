# su

## SYNOPSIS
Windows version of the linux command \`SU\`

## SYNTAX

```
su [<CommonParameters>]
```

## DESCRIPTION
Immitate SU on Linux.
This creates new PoSH Session as an admin.

## EXAMPLES

### EXAMPLE 1
```
su
```

Description
