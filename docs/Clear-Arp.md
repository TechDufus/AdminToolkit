# Clear-Arp

## SYNOPSIS
Use Netsh to clear ArpCache

## SYNTAX

```
Clear-Arp [<CommonParameters>]
```

## DESCRIPTION
Clears the local arp table

## EXAMPLES

### EXAMPLE 1
```
Clear-Arp
```

Description
-----------
This will clear the arpcache on the local machine.
